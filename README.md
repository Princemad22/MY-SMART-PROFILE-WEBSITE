# مشروع ORCA Truck Parts

## نظرة عامة

مشروع ORCA Truck Parts هو موقع إلكتروني احترافي لشركة توريد قطع غيار الشاحنات. تم تطويره باستخدام Laravel (PHP) مع التركيز على:

- دعم لغتين: العربية والإنجليزية
- وضع نهاري ومظلم قابل للتبديل
- واجهة مستخدم احترافية
- لوحة تحكم متكاملة
- أداء عالي وقابلية للتوسع

## المميزات

### 1. التعدد اللغوي
- دعم كامل للغة العربية والإنجليزية
- إمكانية التبديل بين اللغات من الواجهة
- دعم RTL الكامل للغة العربية

### 2. الوضع المظلم والنهاري
- زر تبديل واضح في الواجهة
- حفظ تفضيل المستخدم في localStorage
- دعم كامل لكلا الوضعين مع تباين مناسب

### 3. التصميم المتجاوب
- تصميم متوافق مع جميع الأجهزة
- دعم الشاشات الصغيرة والكبيرة
- تجربة مستخدم ممتازة

### 4. لوحة التحكم
- واجهة إدارية باللغة العربية
- نظام صلاحيات للمستخدمين
- إدارة المحتوى بسهولة

## هيكل المشروع

```
project-root/
├── app/                    # ملفات التطبيق
│   ├── Http/               # طبقات HTTP (متحكمات، وسطاء)
│   ├── Models/             # نماذج قاعدة البيانات
│   └── ...
├── resources/              # الموارد (الواجهات، الترجمات)
│   ├── views/              # قوالب Blade
│   ├── lang/               # ملفات الترجمة
│   └── ...
├── public/                 # الملفات العامة
├── storage/                # الملفات المؤقتة والوسائط
├── routes/                 # ملفات التوجيه
├── config/                 # إعدادات التطبيق
└── ...
```

## الملفات الحالية

- `index.html`: الصفحة الرئيسية للموقع
- `INSTALLATION.md`: دليل التثبيت التفصيلي

## المتطلبات

- PHP 8.1+
- Laravel Framework
- قاعدة بيانات (MySQL/MariaDB)
- Composer
- Node.js وnpm

## التثبيت

يرجى الرجوع إلى ملف `INSTALLATION.md` للحصول على تعليمات التثبيت التفصيلية.

## الاستخدام

### 1. تشغيل الخادم
```bash
php artisan serve
```

### 2. تشغيل واجهة المستخدم
```bash
npm run dev
```

## هيكل الصفحة الرئيسية

الصفحة الرئيسية تتضمن:

1. **الهيدر**
   - شعار ORCA Truck Parts
   - قائمة التنقل
   - زر تبديل اللغة
   - زر تبديل الوضع

2. **القسم الرئيسي (Hero)**
   - صور عالية الجودة
   - عنوان قوي
   - أزرار تحفيزية

3. **قسم عن الشركة**
   - نبذة مختصرة
   - نقاط القوة

4. **قسم الخدمات**
   - قطع غيار مرسيدس
   - قطع غيار سكانيا
   - قطع غيار فولفو

5. **قسم الإحصائيات**
   - سنوات الخبرة
   - عدد العملاء
   - عدد القطع

6. **قسم الشركاء**
   - شعارات الموردين

7. **قسم دعوة للعمل**
   - زر طلب عرض سعر

8. **الفوتر**
   - معلومات الاتصال
   - روابط سريعة
   - وسائل التواصل
